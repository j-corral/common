## Description

Read MDX source as string (could come from file, api,...) then try to convert into a parsed object.
It includes Matter metadata and the unique list of the special components used in the source.

### Function usage

```js
import { ParseMDX } from '@hoomies/fuel.functions.parse-mdx';

const source = '# Hello World !\n I am Happy <Unagui.Hero>toto</Unagui.Hero>';
const expected = { componentNames: ['UnaguiHero'] };
const parsed = ParseMDX({ source });
```
