## Usage

```ts
import { TReactElements } from '@hoomies.fuel.types.treact';


const interface MyInterface {
    List: TReactElements;
};
```

## React Types

|       Name       | Type                   | Description                    |
| :--------------: | ---------------------- | ------------------------------ |
| `TReactElement`  | `React.ReactElement`   | React Element                  |
| `TReactElements` | `Array<TReactElement>` | A collection of React Elements |
|                  |                        |                                |
