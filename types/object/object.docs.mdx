## Usage

```ts
import { TDObject } from '@hoomies/fuel.types.object';

const Test: TDObject = {
  A: 'Hello',
  B: [1, 2, 3],
  D: false,
};
```

## Object Types

|    Name    | Type     | Description                              |
| :--------: | -------- | ---------------------------------------- |
| `TDObject` | `Object` | An object with dynamic string properties |
|            |          |                                          |
